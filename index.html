My Projects

Project 1: Road Traffic Sign Recognition

This computer vision project was developed by me as part of an internship at a software development company. The goal of this project was to recognize different traffic signs on road using image processing deep learning technique. The datasets provided by them consisted of 43 different classes/categories of traffic signs. I was required to create a convolutional neural network model architechture to train and test various images. I was able to achieve an accuracy of 97.07% on the validation set (random 20% subset of the train dataset) & an accuracy of 96.37% on the new/test dataset.
